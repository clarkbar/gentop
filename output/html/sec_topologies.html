Section 2.1 Topologies

There are three observations we can make about our study of subspaces of Euclidean space:

  1. When we think about the topological nature of a subspace \(X \subseteq \RR^n\text{,}\) the ambient Euclidean space in which we find them isn't all that important; homeomorphic subspaces can be found in wildly different Euclidean spaces. The only thing we do want to remember from the ambient \(\RR^n\) is the definition of the closure operator \(\tau_X\text{.}\)

  2. For the vast majority of the work we've done so far, we didn't need the precise definition of \(\tau_X\) in terms of the metric on \(\RR^n\text{.}\) Rather, what we really used repeatedly were the properties given in Proposition 1.1.4. Almost everything we did was just a direct consequence of those!

  3. There are examples of sets with a reasonable notion of closeness that don't arrive in your lap embedded in a Euclidean space. We want to do topology with these examples too!

Example 2.1.1.

Let's illustrate this last point with an interesting example. Let \(\PP^1_{\RR}\) be the set of \(1\)-dimensional linear subspaces of \(\RR^2\text{.}\) In other words, an element of \(\PP^1_{\RR}\) is a line in \(\RR^2\) through the origin. If \(L \in \PP^1_{\RR}\text{,}\) let \(\mu(L)\) denote the slope of \(L\text{;}\) that is, if \(L\) is spanned by a nonzero vector \((x,y) \in \RR^2\text{,}\) then

\begin{equation*} \mu(L) = \begin{cases}y/x \amp \text{ if } x \neq 0\text{;} \\ \infty \amp \text{ if } x=0 . \end{cases} \end{equation*}

We will say that a line \(L \in \PP^1_{\RR}\) is close to a subset \(S \subseteq \PP^1_{\RR}\) if and only if one of the following conditions holds:

  • if \(\mu(L) \neq \infty\text{,}\) then for every \(\varepsilon>0\text{,}\) there exists a line \(L' \in S\text{,}\) such that

    \begin{equation*} |\mu(L) - \mu(L')| \lt \varepsilon\text{,} \end{equation*}

    or

  • if \(\mu(L) = \infty\text{,}\) then for every real number \(N\text{,}\) there exists a line \(L' \in S\) such that

    \begin{equation*} |\mu(L')|>N\text{.} \end{equation*}

Equivalently, if we think of the circle \(S^1 \subset \RR^2\text{,}\) then every line \(L \in \PP^1_{\RR}\) intersects \(S^1\) in exactly two points

For our purposes, it won't matter which of the two intersection points we call \(i(L)\) and which we call \(j(L)\text{.}\)

\(i(L)\) and \(j(L)\text{,}\) where \(i(L) = -j(L)\text{.}\) We may now say that \(L \in \PP^1_{\RR}\) is close to \(S \subseteq \PP^1_{\RR}\) if and only if, for every \(\varepsilon>0\text{,}\) there exists a line \(L' \in S\) such that either \(\|i(L)-i(L')\|\lt \varepsilon\) or \(\|i(L)-j(L)\|\lt \varepsilon\text{.}\)

To deal with examples like this one, we're going to pull a trick that is often quite powerful in mathematics: we're going to turn Proposition 1.1.4 into a definition. Thus a topology on a set \(X\) is a systematic way of asking whether an element of \(X\) — which we will call point — is “close” to a subset. In order to constitute a topology on the set \(X\text{,}\) this notion of closeness must satsify three conditions:

  • For every subset \(S \subseteq X\) and any point \(x \in X\text{,}\) if \(x \in S\text{,}\) then \(x\) is close to \(S\text{.}\) More briefly, an element of a subset is close to that subset.

  • For every finite collection \(\{ S_1, \dots, S_n \}\) of subsets of \(X\text{,}\) if a point \(x \in X\) is close to the union \(S_1 \cup \dots \cup S_n\text{,}\) then there exists \(i \in \{ 1, \dots, n \}\) such that \(x\) is close to \(S_i\text{.}\) More briefly, if a point is close to a finite union of subsets, then it is close to one of those subsets. In particular, when \(n = 0\text{,}\) this condition says that no point is close to the empty set \(\emptyset\text{.}\)

  • Suppose \(S\) and \(T\) are two subsets of \(X\) such that every point \(s \in S\) is close to \(T\text{;}\) then if a point \(x\) is close to \(S\text{,}\) it is also close to \(T\text{.}\)

To formalise this idea, we introduce, for each subset \(S \subseteq X\text{,}\) the set \(\tau(S)\) of all the elements of \(X\) that are close to \(S\text{,}\) and we list its properties.

Definition 2.1.2.

A topology on a set \(X\) is a map

\begin{equation*} \tau \colon \PP(X) \to \PP(X) \end{equation*}

such that the following conditions hold.

  1. For any \(S \in \PP(X)\text{,}\) one has \(S \subseteq \tau(S)\text{.}\) In particular, \(\tau(X) = X\text{.}\)

  2. For any finite subset \(\Sigma \subseteq \PP(X)\text{,}\) one has

    \begin{equation*} \tau \left( \bigcup_{ S \in \Sigma} S \right) = \bigcup_{ S \in \Sigma } \tau(S)\text{.} \end{equation*}

    In particular,

    An empty union \(\bigcup_{S \in \varnothing} Y_S\) is empty.

    \(\tau (\emptyset) = \emptyset\text{.}\) Also, observe that \(\tau\) is inclusion-preserving; that is, if \(S \subseteq T\text{,}\) then \(\tau(S) \subseteq \tau(T)\text{.}\)

  3. For any \(S \in \PP(X)\text{,}\) one has \(\tau( \tau( S ) ) = \tau( S )\text{.}\)

If \(x \in X\) and \(S \in \PP(X)\text{,}\) then we say that \(x\) is close to \(S\) if and only if \(x \in \tau(S)\text{.}\)

A pair \((X, \tau)\) consisting of a set \(X\) and a topology \(\tau\) on \(X\) is called a topological space. We call the elements of a topological space points.

The set \(\tau(S)\) is called the closure of \(S\text{.}\) We say that \(S\) is closed if \(S = \tau( S )\text{,}\) and we say that \(S\) is open if its complement is closed.

In a little while, we'll discuss how one can specify a topology by listing the open sets, which is what you will frequently see in other books.

Example 2.1.3.

We have already seen the first example. Any subset \(X \subseteq \RR^n\) inherits a topology \(\tau_X\) that carries a subset \(S \subseteq X\) to

\begin{equation*} \tau_X(S) = \{ x\in X : (\forall \varepsilon>0)(\exists s \in S)(d(x,s)\lt \varepsilon\}\text{.} \end{equation*}

When \(X=\RR^n\text{,}\) this is often called the standard topology, and for \(X \subseteq \RR^n\text{,}\) this is called the subspace topology.

Example 2.1.4.

The closure operation on \(\PP^1_{\RR}\) (Example 2.1.1) is a topology.

  1. Let \(S \subseteq \PP^1_{\RR}\text{.}\) If \(L \in S\text{,}\) then the conditions are automatic for \(L \in \tau(S)\text{.}\)

  2. Let \(S_1, \dots, S_n \subseteq \PP^1_{\RR}\) be a collection of subsets of \(\PP^1_{\RR}\text{,}\) and let \(L \in \PP^1_{\RR}\text{.}\) If \(L\) is close to \(S_1 \cup \cdots \cup S_n\text{,}\) then a Pigeonhole argument like the one given in Proposition 1.1.4 ensures that \(L\) is close to \(S_1 \cup \cdots \cup S_n\text{.}\)

    Fill in the details here! Often, in writing, mathematicians say something like, “by the same argument …,” or “by the standard argument …,” or even, “it is obvious that….” This can be a little annoying (and I don't recommend calling very many things “obvious”), but it is a good opportunity for you to take a moment to sort through the details yourself.

    If \(L\) is close to some \(S_i\text{,}\) then it is certainly close to \(S_1 \cup \cdots \cup S_n\text{.}\)

  3. Let \(S \subseteq \PP^1_{\RR}\text{,}\) and let \(L \in \PP^1_{\RR}\) be a line that is close to \(\tau(S)\text{.}\) We aim to show that \(L\) is close to \(S\text{;}\) so let \(\varepsilon>0\text{.}\) There exists a line \(L' \in \tau(S)\) such that either \(\|i(L) - i(L')\|\lt \varepsilon\) or \(\|i(L)-j(L')\|\lt \varepsilon/2\text{.}\) Accordingly, there exists a line \(L'' \in S\) such that either \(\|i(L') - i(L'')\| \lt \varepsilon/2\) or \(\|i(L') - j(L'')\| \lt \varepsilon/2\text{.}\) Analyzing the four options and applying the triangle inequality, we find that either \(\|i(L) - i(L'')\|\lt \varepsilon\) or \(\|i(L) - j(L'') \|\lt \varepsilon\text{.}\)

Example 2.1.5.

The empty set \(\emptyset\) admits a unique topology. The powerset \(\PP( \emptyset )\) is a singleton \(\{ \emptyset \}\text{,}\) and the identity is the only map \(\{ \emptyset \} \to \{ \emptyset \}\text{.}\) This map is a topology. The only subset available — \(\emptyset\) itself — is both open and closed.

Example 2.1.6.

More generally, on any set \(X\text{,}\) the identity map

\begin{equation*} \delta \colon \PP(X) \to \PP(X) \end{equation*}

is always a topology. This is called the discrete topology on a set \(X\text{.}\)

In the discrete topology, a point \(x \in X\) is close to a subset \(S \subseteq X\) if and only if \(x \in S\text{.}\) Every subset of \(X\) is both open and closed in the discrete topology.

Example 2.1.7.

Let us study topologies on the singleton \(\{ x \}\text{.}\) The power set \(\PP( \{ x \} )\) is a two-element set \(\{ \emptyset, \{ x \} \}\text{.}\) There are four maps from this set to itself, but most of these are not topologies. In order for a map \(\tau\) to be a topology, the first condition demands that \(\tau( \{ x \} ) = \{ x \}\text{,}\) and the second condition demands that \(\tau ( \emptyset ) = \emptyset\text{.}\) Thus the only topology on the singleton \(\{ x \}\) is the discrete topology.

Example 2.1.8.

Let \(X \coloneq \{ x , y \}\text{.}\) In order to specify a topology \(\tau\) on \(X\text{,}\) it is really only necessary to specify the subsets \(\tau ( \{ x \} )\) and \(\tau ( \{ y \} )\text{.}\) The first condition states that these subsets must contain the sets \(\{ x \}\) and \(\{ y \}\text{,}\) respectively, and the second and third conditions become automatic. That gives us four topologies on \(\{ x, y \}\text{:}\)

  • the discrete topology, in which \(\tau( \{ x \} ) = \{ x \}\) and \(\tau( \{ y \} ) = \{ y \}\text{;}\)

  • the topology in which \(\tau( \{ x \} ) = \{ x \}\text{,}\) and \(\tau( \{ y \} ) = X\text{;}\)

  • the topology in which \(\tau( \{ y \} ) = X\text{,}\) and \(\tau( \{ x \} ) = \{ x \}\text{;}\) and

  • the topology in which \(\tau( \{ x \} ) = \tau( \{ y \} ) = X\text{.}\)

Let us describe the relation of closeness for each of these topologies. The only questions that aren't answered by the axioms of a topology is: “is \(x\) close to \(\{ y \}\text{?}\)” and “is \(y\) close to \(\{ x \}\text{?}\)” Let's answer these in each case:

  • in the discrete topology, \(x\) is not close to \(\{ y \}\text{,}\) and \(y\) is not close to \(\{ x \}\text{;}\)

  • in the second of these topologies, \(x\) is close to \(\{ y \}\text{,}\) but \(y\) is not close to \(\{ x \}\text{;}\)

  • in the third topology, \(y\) is close to \(\{ x \}\text{,}\) but \(x\) is not close to \(\{ y \}\text{;}\) and

  • finally, in the last of these topologies, \(x\) is close to \(\{ y \}\) and \(y\) is close to \(\{ x \}\text{.}\)

The asymmetry in the second and third topologies is strange from our natural idea of what “close” ought to mean. Nevertheless, these topologies are actually interesting examples; even rather bizarre ideas of closeness can be captured in topology!

Finally, for each of these examples, let us determine which sets are open and closed. In each case, the only questions that aren't answered by the axioms are: “is \(\{ x \}\) open? closed?” and “is \(\{ y \}\) open? closed?”

  • in the discrete topology, both \(\{ x \}\) and \(\{ y \}\) are both open and closed;

  • in the second topology, \(\{ x \}\) is closed and not open, and \(\{ y \}\) is open and not closed.

  • in the third topology, the situation is reversed: \(\{ x \}\) is open and not closed, and \(\{ y \}\) is closed and not open;

  • in the last of these topologies, neither \(\{ x \}\) nor \(\{ y \}\) is open or closed.

The discrete topology makes sense for any set. Similarly, the last of the topologies in Example 2.1.8 above is sensible for any set.

Example 2.1.9.

For any set \(X\text{,}\) we define a map

\begin{equation*} \iota \colon \PP(X) \to \PP(X) \end{equation*}

by the formula

\begin{equation*} \iota( S ) \coloneq \begin{cases}\emptyset \amp \text{ if } S = \emptyset ; \\ X \amp \text{ if } S \neq \emptyset . \end{cases} \end{equation*}

This is a topology called the indiscrete topology. Thus in the indiscrete topology, any point is close to any nonempty subset. The only subsets of \(X\) that are open or closed in the indiscrete topology are \(\emptyset\) and \(X\) itself.

The other two topologies of Example 2.1.8 are more interesting:

Example 2.1.10.

Let \(P\) be a preorder. That is, \(P\) is a set equipped with a relation \(\leq\) such that:

  • for every \(a \in P\text{,}\) one has \(a \leq a\text{;}\) and

  • for every \(a,b,c \in P\text{,}\) if \(a \leq b\) and \(b \leq c\text{,}\) then \(a \leq c\text{.}\)

Then we can define a topology as follows. For any subset \(S \subseteq P\text{,}\) we define

\begin{equation*} \tau_{\leq}(S) \coloneq \{a \in P : (\exists s \in S)(a \leq s) \}\text{.} \end{equation*}

In other words, \(a \in P\) is close to \(S\) if and only if some element of \(S\) exceeds it.

Let us quickly confirm that this is a topology:

  1. Let \(S \subseteq P\) be a subset. Since \(s \leq s\) for every \(s \in S\text{,}\) it follows that \(S \subseteq \tau_{\leq}(S)\text{.}\)

  2. Let \(S_1, \dots, S_n \subseteq P\) be a finite collection of subsets of \(P\text{.}\) Then

    \begin{align*} \tau_{\leq}(S_1 \cup \cdots \cup S_n) \amp = \{a\in P : (\exists s \in S_1 \cup \cdots \cup S_n)(a \leq s) \}\\ \amp = \{a \in P : (\exists i)(\exists s \in S_i)(a \leq s)\}\\ \amp = \tau_{\leq}(S)_1 \cup \cdots \cup \tau_{\leq}(S)_n \text{.} \end{align*}

  3. Let \(S \subseteq P\) be a subset. Then

    \begin{align*} \tau_{\leq}(\tau_{\leq}(S)) \amp = \{a \in P : (\exists t \in \overline{S})(a \leq t) \}\\ \amp = \{a \in P : (\exists s \in S)(\exists t \in X)(a \leq t \leq s) \}\\ \amp = \{a \in P : (\exists s \in S)(a \leq s) \}\\ \amp = \tau_{\leq}(S) \text{.} \end{align*}

This topology is called the Alexandroff topology.

The closed subsets are the subsets \(Z \subseteq P\) with the property that if \(a \in P\text{,}\) \(z \in Z\text{,}\) and \(a \leq z\text{,}\) then \(a \in Z\) as well. The open subsets are the subsets \(U \subseteq P\) with the property that if \(a \in P\text{,}\) \(u \in U\text{,}\) and \(a \geq u\text{,}\) then \(a \in U\) as well.

There is another , more standard, way to specify topologies. It's a little less intuitive, but it becomes technically convenient when we want to do things like generate topologies.

Definition 2.1.13.

Fix a set \(X\text{.}\) Let's define two different kinds of information on \(X\text{.}\)

  • A system of open sets on \(X\) is a subset \(\mathscr{O}\subseteq\PP(X)\) that is stable under unions

    possibly empty

    and finite intersections

    possibly empty

    . That is, if \(\Sigma\subseteq \mathscr{O}\text{,}\) then the union

    \begin{equation*} \bigcup_{U\in \Sigma} U \end{equation*}

    also lies in \(\mathscr{O}\text{,}\) and if \(\Sigma\) is finite, then the intersection

    \begin{equation*} \bigcap_{U\in \Sigma} U \end{equation*}

    also lies in \(\mathscr{O}\text{.}\)

  • Dually, a system of closed sets on \(X\) is a subset \(\mathscr{C}\subseteq\PP(X)\) that is stable under intersections and finite unions. That is, if \(\Sigma \subseteq\mathscr{C}\text{,}\) then the intersection

    \begin{equation*} \bigcap_{Z\in\Sigma} Z \end{equation*}

    also lies in \(\mathscr{C}\text{,}\) and if \(\Sigma\) is finite, then the union

    \begin{equation*} \bigcup_{Z\in\Sigma} Z \end{equation*}

    also lies in \(\mathscr{C}\text{.}\)

Specifying a system of open sets is the same as specifying a system of closed sets. More precisely, the formation of the complement in \(X\) defines a bijection between the set of systems of open sets and the set of systems of closed sets.

Specifying a topology is the same as specifying a system of closed sets.

For any a topology \(\tau\) on \(X\text{,}\) one may define \(\mathscr{C}_{\tau}\) as the set of closed subsets of the topology:

\begin{equation*} \mathscr{C}_{\tau} \coloneq \{ Z\in\PP(X) : \tau(Z)=Z \}\text{.} \end{equation*}

To see that \(\mathscr{C}_{\tau}\) is stable under intersection and finite union, let \(\Sigma \subseteq \mathscr{C}_{\tau}\) be a subset. Let \(W\) be the intersection \(\bigcap_{Z \in \Sigma} Z\text{.}\) For every \(Z \in \Sigma\text{,}\) since \(\tau\) is inclusion-preserving, it follows that \(\tau\left(W\right) \subseteq \tau(Z) = Z\text{.}\) Consequently, \(\tau(W)\) is contained in the intersection \(W\text{,}\) so \(W\) is closed. If \(\Sigma\) is finite, then since \(\tau\) preserves finite unions, we have

\begin{equation*} \tau\left(\bigcup_{Z\in\Sigma} Z\right) = \bigcup_{Z\in\Sigma} \tau(Z) = \bigcup_{Z \in \Sigma} Z\text{.} \end{equation*}

This proves that \(\mathscr{C}_{\tau}\) is a system of closed sets.

In the opposite direction, for any system of closed subsets \(\mathscr{C}\) on \(X\text{,}\) define a map \(\tau_{\mathscr{C}}\colon\PP(X) \to \PP(X)\) that carries a subset \(S\subseteq X\) to the smallest element of \(\mathscr{C}\) that contains \(S\text{:}\)

Since \(\mathscr{C}\) is stable under intersections, this formula indeed defines an element of \(\mathscr{C}\text{.}\) So \(\tau_{\mathscr{C}}(S)\) is the smallest element of \(\mathscr{C}\) that contains \(S\text{.}\)

\begin{equation*} \tau_{\mathscr{C}}(S) \coloneq \bigcap_{\substack{Z\in \mathscr{C},\\ S\subseteq Z} } Z\text{.} \end{equation*}

Let us see that \(\tau_{\mathscr{C}}\) is a topology.

  1. By definition, for every subset \(S \subseteq X\text{,}\) we have \(S \subseteq \tau(S)\text{.}\)

  2. By definition, \(\tau_{\mathscr{C}}\) is an inclusion-preserving operation. Thus if \(S_1, \dots, S_n \subseteq X\) is a finite collection of subsets of \(X\text{,}\) then it follows that

    \begin{equation*} \tau_{\mathscr{C}}(S_1 \cup \cdots \cup S_n) \supseteq \tau_{\mathscr{C}}(S_1) \cup \cdots \cup \tau_{\mathscr{C}}(S_n)\text{.} \end{equation*}

    On the other hand, since \(\mathscr{C}\) is stable under finite unions, it follows that \(\tau_{\mathscr{C}}(S_1) \cup \cdots \cup \tau_{\mathscr{C}}(S_n) \in \mathscr{C}\text{.}\) Since in addition, \(S_1 \cup \cdots \cup S_n \subseteq \tau_{\mathscr{C}}(S_1) \cup \cdots \cup \tau_{\mathscr{C}}(S_n)\text{,}\) it follows

    The point here is that \(\tau_{\mathscr{C}}(S_1 \cup \cdots \cup S_n)\) is the smallest closed subset that contains \(S_1 \cup \cdots \cup S_n\text{.}\)

    that

    \begin{equation*} \tau_{\mathscr{C}}(S_1 \cup \cdots \cup S_n) \subseteq \tau_{\mathscr{C}}(S_1) \cup \cdots \cup \tau_{\mathscr{C}}(S_n)\text{,} \end{equation*}

    whence \(\tau_{\mathscr{C}}\) preserves finite unions.

  3. Finally, \(\tau_{\mathscr{C}}(\tau_{\mathscr{C}}(S)\) is the smallest element of \(\mathscr{C}\) that contains \(\tau_{\mathscr{C}}(S)\text{.}\) But since \(\tau_{\mathscr{C}}(S)\) itself likes in \(\mathscr{C}\text{,}\) it follows that

Thus one may specify a topology on \(X\) by specifying the closed sets and checking stability under intersections and finite unions, or specifying the open sets and checking stability under unions and finite intersections.

Most textbooks define topologies as a system of open sets. There's nothing wrong with this definition, but we've delayed this description, because it seems difficult to motivate fully.

Example 2.1.14.

Here's an extremely important example in modern mathematics. This one will be following us around throughout this text. Let \(X \subseteq \RR^m\) be any subspace, and let \(n \in \NN\text{.}\) We are about to define a topological space \(C_n(X)\) called the configuration space of \(n\) points in \(X\). As a set, \(C_n(X)\) is the set of subsets \(\{x_1,\dots,x_n\} \subseteq X\) of \(n\) distinct points:

\begin{equation*} C_n(X) \coloneq \{ S \subseteq \PP(X) : \# S = n \}\text{.} \end{equation*}

To endow it with a topology, we need to define an auxiliary topological space.

Recall that we defined the subspace

\begin{equation*} X^n \coloneq X \times \cdots \times X = \{ x = (x_1, \dots, x_n) \in \RR^{mn} : (\forall i)(x_i \in X) \} \subseteq \RR^{mn}\text{.} \end{equation*}

We now pass to a further subspace:

\begin{equation*} E_n(X) \coloneq \{(x_1, \dots, x_n) \in X^n : (\forall i,j)((i \neq j) \implies (x_i \neq x_j)) \} \subseteq X^m\text{.} \end{equation*}

Now there is a map \(q \colon E_m(X) \to C_m(X)\) defined by

\begin{equation*} q(x_1, \dots, x_n) \coloneq \{x_1, \dots, x_n\}\text{.} \end{equation*}

We now declare that \(U \subseteq C_n(X)\) is open if and only if \(q^{-1}(U) \subseteq E_n(X)\) is open. This defines a system of open sets (and hence a topology), since the formation of the inverse image preserves unions and intersections.

Definition 2.1.16.

Let \(X\) be a topological space, and let \(x \in X\text{.}\) Then an open neighborhood of \(x\) is an open subset \(U \subseteq X\) such that \(x \in U\text{.}\) A neighborhood of \(x\) is a subset of \(X\) that contains an open neighborhood of \(x\text{.}\)

If \(X\) is a topological space, then \(\tau(S)\) is the smallest closed subset that contains \(S\text{.}\) Equivalently, a point \(y \in X\) lies in the closure \(\tau(S)\) if and only if, for any open neighborhood \(U\) of \(y\text{,}\) the intersection \(U \cap \tau(S)\) is nonempty.

Why are these equivalent? Well, if \(y \notin \tau(S)\text{,}\) then \(X \smallsetminus \tau(S)\) is an open neighborhood of \(y\) that does not intersect \(S\text{.}\) Conversely, if \(U\) is an open neighborhood of \(x\) that does not intersect \(\tau(S)\text{,}\) then \(X \smallsetminus U\) is a closed subset that contains \(S\) and therefore \(\tau(S)\text{.}\)
