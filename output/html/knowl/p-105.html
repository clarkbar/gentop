Let \(X \subseteq \RR^m\) and \(Y \subseteq \RR^n\) be subspaces. Then a homeomorphism \(f \colon X \to Y\) is a continuous bijection whose inverse \(f^{-1}\) is continuous. In other words, a map \(f \colon X \to Y\) is a homeomorphism if and only if \(f\) is a bijection such that for any subset \(S \subseteq X\text{,}\) one has \(\tau_Y(f(S)) = f(\tau_X(S))\text{.}\) Equivalently, \(f \colon X \to Y\) is a homeomorphism if and only if it is a bijection such that both the image and the inverse image of closed (respectively, open) subsets remains closed (resp., open).
