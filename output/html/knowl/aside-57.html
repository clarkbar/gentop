The phrase “there is at most one \(x\) such that \(\phi(x)\)” is a helpful shorthand for the longer sentence

\begin{equation*} (\forall x)(\forall y)((\phi(x)\wedge\phi(y))\implies(x=y))\text{.} \end{equation*}

This is a pretty annoying turn of language, one has to admit: we're actually using the phrase “there is” in a sentence in which the only quantifier is universal.
