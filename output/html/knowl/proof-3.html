Proof

Let \(Z \subseteq X\) be a closed subset that contains \(S\text{.}\) Then

\begin{equation*} \tau_X(S) \subseteq \tau_X(Z) = Z\text{.} \end{equation*}

Hence \(\tau_X(S)\) is contained in any closed subset of \(X\) that contains \(S\text{.}\) It remains to note that \(\tau_X(S)\) is itself a closed subset of \(X\) that contains \(S\text{.}\) By the proposition above,

\begin{equation*} \tau_X(\tau_X(S)) = \tau_X(S)\text{,} \end{equation*}

so \(\tau_X(S)\) is closed.
