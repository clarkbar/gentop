Example 2.1.12.

Define a topology on \(\RR\sqcup\{-\infty,+\infty\}\) in which, for any point \(x\in\RR\sqcup\{-\infty,+\infty\}\) and any subset \(S\subseteq\RR\sqcup\{-\infty,+\infty\}\text{,}\)

  • If \(x\in\RR\text{,}\) then \(x\) is close to \(S\) if and only if \(x\) is close to \(S\cap\RR\text{.}\)

  • We declare that \(-\infty\) is close to \(S\) if and only if either \(-\infty\in S\) or for any \(N\in\RR_{\geq 0}\text{,}\) there is a point \(s\in S\cap\RR\) such that \(s\lt -N\text{.}\)

  • We declare that \(+\infty\) is close to \(S\) if and only if either \(+\infty\in S\) or for any \(N\in\RR_{\geq 0}\text{,}\) there is a point \(s\in S\cap\RR\) such that \(s>N\text{.}\)

This is a topology.

You should convince yourself of this!
