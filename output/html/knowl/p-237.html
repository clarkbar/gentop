Let us study topologies on the singleton \(\{ x \}\text{.}\) The power set \(\PP( \{ x \} )\) is a two-element set \(\{ \emptyset, \{ x \} \}\text{.}\) There are four maps from this set to itself, but most of these are not topologies. In order for a map \(\tau\) to be a topology, the first condition demands that \(\tau( \{ x \} ) = \{ x \}\text{,}\) and the second condition demands that \(\tau ( \emptyset ) = \emptyset\text{.}\) Thus the only topology on the singleton \(\{ x \}\) is the discrete topology.
