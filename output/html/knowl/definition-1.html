Definition 1.1.1.

Let \(S \subseteq \RR^n\) be a subset. A point \(x \in \RR^n\) will be said to be close to \(S\) if and only if, for every \(\varepsilon > 0\text{,}\) there exists an element \(s \in S\) such that \(d(x,s) \lt \varepsilon\text{.}\)

We may express this notion with the aid of an auxiliary quantity. Define the distance from \(x\) to \(S\) as

If \(S = \varnothing\text{,}\) then this infimum doesn't exist as a real number. So we'll declare formally that \(d(x,\varnothing) = +\infty\text{.}\)

\begin{equation*} d(x,S) \coloneq \inf \{ d(x,s) : s \in S \}\text{.} \end{equation*}

Then \(x\) is close to \(S\) if and only if \(d(x,S) =0\text{.}\)

Let \(X \subseteq \RR^n\) be a subset such that \(S \subseteq X\text{.}\) We shall write \(\tau_X(S) \subseteq X\) for the set of \(x \in X\) that are close to \(S\text{:}\)

\begin{equation*} \tau_X(S) \coloneq \{ x \in X : (\forall \varepsilon >0)(\exists s \in S)(d(x,s)\lt \varepsilon) \} = \{x \in X : d(x,S) = 0\} \end{equation*}

This is called the closure of \(S\) in \(X\). In other words,

\begin{equation*} \tau_X(S) = X \cap \bigcap_{\varepsilon>0}\bigcup_{s\in S} B^n(s,\varepsilon)\text{.} \end{equation*}

We will refer to \(X\) along with the map

The set \(\PP(X)\) is the powerset of \(X\text{;}\) it is the set of subsets of \(X\text{.}\)

\(\tau_X \colon \PP(X) \to \PP(X)\) as the subspace \(X \subseteq \RR^n\text{.}\)
