Example 1.1.2.

  • If \(X = \varnothing\text{,}\) then this definition becomes a little boring: the only subset of \(\varnothing\) is \(\varnothing\) itself, and \(\tau_{\varnothing}(\varnothing) = \varnothing\text{.}\)

  • If \(X = \RR\) itself, then let's see what happens for various choices of \(S\text{:}\)

    • Let's consider various unit intervals:

      \begin{equation*} \tau_{\RR}(\left]0,1\right[) = \tau_{\RR}([0,1]) = \tau_{\RR}(\left]0,1\right]) = [0,1] \end{equation*}

    • If \(\QQ \subset \RR\) denotes the set of rational numbers, then \(\tau_{\RR}(\QQ) = \RR\text{.}\)

    • At the same time, \(\tau_{\RR}(\RR \smallsetminus \QQ) = \RR\text{.}\)

    • Confirm that \(\tau_{\RR}(\varnothing)=\varnothing\text{.}\)

  • If \(X \subseteq \RR^n\) is a subspace, then we have

    \begin{equation*} \tau_X(S) = \tau_{\RR^n}(S) \cap X\text{.} \end{equation*}

    This is an important point: the closure of \(S\) is done relative to the subspace \(X\). Let's see how the closure of \(S = \left]0,1\right[\) depends on which subspace \(X \subseteq \RR\) in which we are closing:

    • If \(X=\RR\text{,}\) then \(\tau_{\RR}(\left]0,1\right[) = [0,1]\text{.}\)

    • If \(X=\left]0,1\right[\text{,}\) then \(\tau_{\left]0,1\right[}(\left]0,1\right[) = \left]0,1\right[\text{.}\)

    • If \(X=\left]0,1\right]\text{,}\) then \(\tau_{\left]0,1\right]}(\left]0,1\right[) = \left]0,1\right]\text{.}\)

    • If \(X=\left]0,+\infty\right[\text{,}\) then \(\tau_{\left]0,+\infty\right[}(\left]0,1\right[) = \left]0,1\right]\text{.}\)
