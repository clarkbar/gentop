Proposition 1.2.2.

Let \(X \subseteq \RR^m\) and \(Y \subseteq \RR^n\) be subspaces. The following are equivalent for a map \(f \colon X \to Y\text{.}\)

  1. The map \(f\) is continuous.

  2. For any subset \(T \subseteq Y\text{,}\) one has \(\tau_X(f^{-1}(T)) \subseteq f^{-1}(\tau_Y(T))\text{.}\)

  3. For any closed subset \(Z \subseteq Y\text{,}\) the inverse image \(f^{-1}(Z) \subseteq X\) is closed.

  4. For any open subset \(U \subseteq Y\text{,}\) the inverse image \(f^{-1}(U) \subseteq X\) is open.

  5. For any \(x \in X\) and any \(\varepsilon > 0\text{,}\) there exists \(\delta >0\) such that if \(x_0 \in X\) is a point such that \(d(x_0,x) \lt \delta\text{,}\) then
