Definition 1.2.1.

Let \(X \subseteq \RR^m\) and \(Y \subseteq \RR^n\) be subspaces. A map \(f \colon X \to Y\) is continuous if and only if, for any subset \(S \subseteq X\) and any point \(x\in X\text{,}\) if \(x\) is close to \(S\text{,}\) then \(f(x)\) is close to \(f(S)\text{.}\) In other words, \(f\) is continuous if and only if, for any subset \(S \subseteq X\text{,}\) one has \(f(\tau_X(S)) \subseteq \tau_Y(f(S))\text{.}\)
