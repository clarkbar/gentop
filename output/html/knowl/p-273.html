Let's see that this is a topology. In effect, you check everything by checking the two cases (\(x\in\RR\) or \(x\notin\RR\)) separately. In the first case, you use the fact that \(\RR\) has a topology already, and in the second, you give a little argument.

  • By definition if \(x\) lies in a subset \(S\subseteq\RR\sqcup\{\infty\}\text{,}\) then \(x\) is close to \(S\text{.}\)

  • Let \(\{S_1,\dots,S_n\}\) be a finite collection of subsets of \(\RR\sqcup\{\infty\}\text{.}\) If \(x\in \RR\text{,}\) then since one has

    \begin{equation*} (S_1\cup\dots\cup S_n)\cap\RR=(S_1\cap\RR)\cup\dots\cup (S_n\cap\RR)\text{,} \end{equation*}

  • Finally, suppose that every point of a subset \(S\subseteq\RR\sqcup\{\infty\}\) is close to a subset \(T\subseteq\RR\sqcup\{\infty\}\text{.}\) Now if \(x\) is close to \(S\text{,}\) there are two options. First, if \(x\in\RR\text{,}\) then \(x\) is close to \(S\cap\RR\) and hence to \(T\cap\RR\text{.}\) Otherwise, if \(x=\infty\text{,}\) then if \(\infty\in T\) or \(\infty\in S\text{,}\) then we're done. If not, then for every \(N\in\RR\text{,}\) there is a point \(s_N\in S\subseteq\RR\) such that \(|s_N|>N\text{;}\) this point \(s_N\) is close to \(T\text{,}\) so if we set \(\epsilon_N\coloneq|s_N|-N\text{,}\) then there exists a point \(t_N\in T\) such that \(|s_N-t_N|\lt \epsilon_N\text{.}\) Thus \(|t_N|>N\text{,}\) and so \(\infty\) is close to \(T\text{.}\)
