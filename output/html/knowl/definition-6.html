Definition 2.1.2.

A topology on a set \(X\) is a map

\begin{equation*} \tau \colon \PP(X) \to \PP(X) \end{equation*}

such that the following conditions hold.

  1. For any \(S \in \PP(X)\text{,}\) one has \(S \subseteq \tau(S)\text{.}\) In particular, \(\tau(X) = X\text{.}\)

  2. For any finite subset \(\Sigma \subseteq \PP(X)\text{,}\) one has

    \begin{equation*} \tau \left( \bigcup_{ S \in \Sigma} S \right) = \bigcup_{ S \in \Sigma } \tau(S)\text{.} \end{equation*}

    In particular,

    An empty union \(\bigcup_{S \in \varnothing} Y_S\) is empty.

    \(\tau (\emptyset) = \emptyset\text{.}\) Also, observe that \(\tau\) is inclusion-preserving; that is, if \(S \subseteq T\text{,}\) then \(\tau(S) \subseteq \tau(T)\text{.}\)

  3. For any \(S \in \PP(X)\text{,}\) one has \(\tau( \tau( S ) ) = \tau( S )\text{.}\)

If \(x \in X\) and \(S \in \PP(X)\text{,}\) then we say that \(x\) is close to \(S\) if and only if \(x \in \tau(S)\text{.}\)

A pair \((X, \tau)\) consisting of a set \(X\) and a topology \(\tau\) on \(X\) is called a topological space. We call the elements of a topological space points.

The set \(\tau(S)\) is called the closure of \(S\text{.}\) We say that \(S\) is closed if \(S = \tau( S )\text{,}\) and we say that \(S\) is open if its complement is closed.

In a little while, we'll discuss how one can specify a topology by listing the open sets, which is what you will frequently see in other books.
