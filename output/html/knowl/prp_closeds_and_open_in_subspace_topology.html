Proposition 1.1.10.

Let \(X \subseteq \RR^n\) be a subspace. Then a subset \(Z \subseteq X\) is closed if and only if \(Z = Z' \cap X\) for some closed subset \(Z' \subseteq \RR^n\text{.}\) Similarly, a subset \(U \subseteq X\) is open if and only if \(U = U' \cap X\) for some open subset \(U' \subseteq \RR^n\text{.}\)
