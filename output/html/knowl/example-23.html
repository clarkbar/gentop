Example 2.1.9.

For any set \(X\text{,}\) we define a map

\begin{equation*} \iota \colon \PP(X) \to \PP(X) \end{equation*}

by the formula

\begin{equation*} \iota( S ) \coloneq \begin{cases}\emptyset \amp \text{ if } S = \emptyset ; \\ X \amp \text{ if } S \neq \emptyset . \end{cases} \end{equation*}

This is a topology called the indiscrete topology. Thus in the indiscrete topology, any point is close to any nonempty subset. The only subsets of \(X\) that are open or closed in the indiscrete topology are \(\emptyset\) and \(X\) itself.
