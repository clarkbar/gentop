Any set \(X\) can be given the cofinite topology, in which a subset \(Z\subseteq X\) is declared to be closed if and only if either \(Z\) is finite or \(Z=X\text{.}\) Thus in the cofinite topology,

\begin{equation*} \tau(S) = \begin{cases}S \amp \text{ if } S \text{ is finite; } \\ X \amp \text{ if } S \text{ is infinite. } \end{cases} \end{equation*}
